Sobre o projeto

O Histórias em Rotas nasce como um hub para reunir roteiros autorais, combinando viagem, cultura, gastronomia, paisagens e criação de conteúdo.

Como usar este site

Publique cada roteiro em um repositório do GitHub Pages e coloque o link no botão correspondente. Assim, este hub vira sua página principal para Instagram, TikTok, WhatsApp e futuras parcerias.

Exemplo: usuario.github.io/roteiro-curitiba-foz
